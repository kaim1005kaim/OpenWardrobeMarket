# Open Wardrobe Market MVP

Pinterest-like fashion marketplace with AI-powered generation.

## Features

- **Browse**: Pinterest-style masonry layout (3/5 columns)
- **Trend**: Trending items sorted by likes
- **Create**: 4-step generation flow with presets and personalization
- **Detail Modal**: Full item view with similar recommendations

## Tech Stack

- Vite + React + TypeScript + Tailwind CSS
- Vitest for testing
- ESLint + Prettier for code quality

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm run test

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

- `src/app/` - Main application code
- `src/app/components/` - UI components
- `src/app/lib/` - Core logic and utilities
- `tests/` - Test files
- `public/` - Static assets

## APIs

- Imagine API for image generation
- DeepSeek API for AI functionality
